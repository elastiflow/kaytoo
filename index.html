Kaytoo Helm chart repository

index.yaml releases source
This GitHub Pages site hosts the Helm repository index for Kaytoo.

Install

helm repo add kaytoo https://elastiflow.github.io/kaytoo
helm repo update

helm upgrade --install kaytoo kaytoo/kaytoo \
  --namespace elastiflow --create-namespace

Tip: use -f values.yaml for overrides instead of long --set lists.
